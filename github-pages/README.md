# 八幡市バドミントンクラブ ひまわり・コスモス・なでしこ

GitHub Pagesで公開するWebサイト本体です。

## 公開対象

このフォルダ内のファイルだけを公開リポジトリへ入れます。

| ファイル/フォルダ | 内容 |
| --- | --- |
| `index.html` | トップページ。 |
| `admin.html` | 管理者ページ。 |
| `css/` | スタイルシート。 |
| `js/` | Firebase連携と画面処理。 |
| `assets/` | 画像などの静的ファイル。 |

## 注意

FirebaseのFirestore Rulesや設計書、メモはこのフォルダに入れません。
